Game Description

This is a two player game. read the following instructions

  1. the game starts with player 1
  2. the dice starts from 1 to 6
  3. the first player can roll as many times as possible as long as the value of the dice is not 1
  4. if the dice value is 1, all the values in the current score is deleted, and the player is switched to the next player
  5. the player can decide to hold the current score that will add up to his high score
  6. any player that reach 100 first wins the game
